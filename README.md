# Engenharia de Dados

## Como Usar
1. Subir os containers usando docker compose
2. Iniciar servidor kafka
3. Criar tópico kafka
4. Setar variáveis de ambiente
5. Subir airflow (airflow init e docker compose)
6. Executar dags

#### Variáveis de ambiente: 
```sh
PSQL_PGUSER
PSQL_PGPASSWORD
PSQL_DBNAME
MONGO_URI
MONGO_DB

```
